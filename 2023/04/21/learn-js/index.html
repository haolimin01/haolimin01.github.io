由来

学习JavaScript最初是因为实验室网站需要维护，之前开发的同学要毕业了，我承担了维护实验室网页的任务。后面陆陆续续学习过几次，都是了解了一些粗浅的语法内容，没有坚持下来。想着我也是可能毕业的，就把之前要做但是没有完成的东西了结一下。正好借着博客平台，督促自己记录学习过程，也提高一下我薄弱的书写水平。

原则

本着费曼学习法的原则，把复杂的东西用简单的语言描述清楚，我把这个系列教程称之为三三也能学会的JavaScript。每天学习一点点，幸福生活多一点。

目的

近来, Electron 框架兴起，很多大型应用使用该框架开发，such as Atom, VS Code, Skype. Electron 使用 HTML, CSS, JS 开发，也就是之前用来做前端的语言现在可以直接开发桌面应用，而且拥有图形界面。我对 Qt 有一点熟悉，总感觉用Qt开发出来的界面有点丑陋，也可能是本人的水平有限。本着一瓶子不满半瓶子晃荡的心理，我想尝试使用 Electron 开发桌面应用，算是自己的兴趣吧。Qt 和 Electron 都是跨平台的开发框架，这一点我非常看好，当然我看好也没用，仅仅是感兴趣 (:-D)

准备工作

  • 参考书籍：D. Flanagan and 李松峰, JavaScript权威指南, 7th ed. 机械工业出版社, 2021.
  • 操作系统：macOS, MacBook Pro 2021 with M1 Pro Chip.
  • 浏览器：Microsoft Edge & Google Chrome.
  • 代码编辑器：Visual Studio Code & Neovim & Sublime Text.
  • JavaScript 版本：ES6

学习记录

Friday, 21 April, 2023

  • 参考网站：MDN, 服务器端Node.js.
  • 浏览器是JavaScript最早的宿主环境，也是JavaScript代码最常见的运行环境。2010年以后，JavaScript代码又有了另一个宿主环境，Node给与了JavaScript访问整个操作系统的权限。
  • 运行 hello world:

    console.log('hello world!')
    使用 node 运行：node hello.js
    使用浏览器运行：创建新文件 hello.html:<script src='hello.js'></script>，使用浏览器加载 hello.html: file://yourpath/hello.html

Tuesday, 25 April, 2023

  • JavaScript 区分大小写，JavaScript 忽略程序记号(token)之间的空格。
  • JS 支持两种注释，单行注释//, 多行注释 /* */
  • JS 使用;分隔语句，但是分号不是必须的。
  • JS 类型可以分为两种：原始类型(数值，字符串，布尔值)和对象类型。JS中的 null, undefined 也是原始值。任何不是数值、字符串、布尔值、符号、null、undefined 值都是对象。
  • JS 与静态语言更大的差别在于，函数和类不仅仅是语言的语法，它们本身就是可以被JS程序操作的值，函数和类也是特殊的对象。
  • JS 解释器会执行自动垃圾收集，不用关心对象或其他值的析构与释放。
  • JS 的对象类型是可修改的 (mutable)，而它的原始类型是不可修改的 (immutable)。
  • 数值：整数字面量，浮点字面量。
  • 字符串
  • 布尔值：true, false
  • JS 有两个操作符用于测试两个值是否相等。一个是严格相等操作符===，如果两个值不是同一种类型，那么这个操作符就不会判定它们相等。但由于JavaScript在类型转换上很灵活，所以==操作符会做类型转换。
  • 使用 let 和 const 声明变量：let用于声明变量，const用于声明常量。
  • 解构赋值：ES6实现了一种复合声明与赋值语法，叫做解构赋值(destructuring assignment)。在解构赋值中，等号右端的值是数组或者对象(“结构化”的值)，而左端通过模拟数组或对象字面量语法指定一个或多个变量。在解构赋值发生时，会从右侧的值提取(解构)出一个或多个值，并保存到左侧列出的变量中。
  • 解构赋值可能最常用在 const, let等声明语句的初始化变量，但也可以在常规赋值表达式中使用。

Wednesday, 26 April, 2023

  • 数组初始化：let matrix = [[1,2,3], [4,5,6], [7,8,9]]
  • 对象初始化：let p = {x:2.3, y:-1.2}
  • 函数定义表达式：函数定义表达式定义JS函数，其值为新定义的函数, let squre = function(x) {return x*x;}
  • 属性访问表达式：expression.identifier, expression[expression]
  • 条件式属性访问：expression?.identifier, expression?.[expression],先判断前面的 expression 是否有效。
  • 调用表达式：调用表达式是JS调用(或执行)函数或方法的一种语法。fun_name(arg1, arg2, ..., argn)
  • 条件式调用：在ES2020中，可以使用?.()来调用函数，先判断前面的参数是否有效。
  • 对象创建表达式：对象创建表达式创建一个新对象并且调用一个函数(称为构造函数)来初始化这个新对象 new Object()
  • 特殊的操作符：in, instanceof, eval(), typeof, delete
  • 条件语句：if, if else, else if, switch
  • 循环语句：while, do/while, for, for/of, for/in
  • 跳转语句：break, continue, return, yield, try/catch/finally, debugger
  • import and export: import和export声明共同用于让一个JS模块中定义的值可以在另一个模块中使用。一个模块就是一个JS代码文件，有自己的全局作用域，完全与其他模块无关。如果要在一个模块中使用另一个模块中定义的值(如函数或类)，唯一的方式就是在定义值的模块中使用export导出值，在使用值的模块中使用import导入值。

Friday, 28 April, 2023

  • 对象是一个复合值,它汇聚多个值并允许我们按名字存储和获取这些值. 对象不仅仅是简单的字符串到值的映射, 除了维持自己的属性之外, JS 对象也可以从其他对象继承属性,这个其他对象称为其”原型”. 对象的方法通常是继承来的属性, 而这种”原型式继承”是JS的主要特性.
  • JS对象是动态的, 可以动态添加和删除属性. 区分直接定义在对象上的属性和那些从原型对象上继承的属性很重要. JS使用术语”自有属性”指代非继承属性.
  • 对象可以通过对象字面量, new关键字, Object.create() 函数来创建.
  • 对象字面量: let empty={}, let point={x:0, y:0}. 对象字面量是一个表达式, 每次求值都会创建并初始化一个新的, 不一样的对象.
  • 使用new创建对象, 以这种方式使用的函数被称为构造函数 (constructor), 目的是初始化新创建的对象. let o = new Object(); let a = new Array().
  • Object.create() 用于创建一个新对象, 使用其第一个参数作为新对象的原型;
  • 数组:JS数组是无类型限制的, 即数组中的元素可以是任意类型, 同一数组的不同元素也可以是不同的类型.
  • 创建数组: 数组字面量, 对可迭代对象使用…扩展操作符, Array() 构造函数, Array() 构造函数, 工厂方法Array.of()和Array.from()
