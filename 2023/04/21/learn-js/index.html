由来

学习JavaScript最初是因为实验室网站需要维护，之前开发的同学要毕业了，我承担了维护实验室网页的任务。后面陆陆续续学习过几次，都是了解了一些粗浅的语法内容，没有坚持下来。想着我也是可能毕业的，就把之前要做但是没有完成的东西了结一下。正好借着博客平台，督促自己记录学习过程，也提高一下我薄弱的书写水平。

原则

本着费曼学习法的原则，把复杂的东西用简单的语言描述清楚，我把这个系列教程称之为三三也能学会的JavaScript。每天学习一点点，幸福生活多一点。

准备工作

  • 参考书籍：D. Flanagan and 李松峰, JavaScript权威指南, 7th ed. 机械工业出版社, 2021.
  • 操作系统：macOS, MacBook Pro 2021 with M1 Pro Chip.
  • 浏览器：Microsoft Edge & Google Chrome.
  • 代码编辑器：Visual Studio Code & Neovim & Sublime Text.
  • JavaScript 版本：ES6

学习记录

Friday, 21 April, 2023

  • 参考网站：MDN, 服务器端Node.js.
  • 浏览器是JavaScript最早的宿主环境，也是JavaScript代码最常见的运行环境。2010年以后，JavaScript代码又有了另一个宿主环境，Node给与了JavaScript访问整个操作系统的权限。
  • 运行 hello world:

    console.log('hello world!')
    使用 node 运行：node hello.js
    使用浏览器运行：创建新文件 hello.html:<script src='hello.js'></script>，使用浏览器加载 hello.html: file://yourpath/hello.html

Tuesday, 25 April, 2023

  • JavaScript 区分大小写，JavaScript 忽略程序记号(token)之间的空格。
  • JS 支持两种注释，单行注释//, 多行注释 /* */
  • JS 使用;分隔语句，但是分号不是必须的。
  • JS 类型可以分为两种：原始类型(数值，字符串，布尔值)和对象类型。JS中的 null, undefined 也是原始值。任何不是数值、字符串、布尔值、符号、null、undefined 值都是对象。
  • JS 与静态语言更大的差别在于，函数和类不仅仅是语言的语法，它们本身就是可以被JS程序操作的值，函数和类也是特殊的对象。
  • JS 解释器会执行自动垃圾收集，不用关心对象或其他值的析构与释放。
  • JS 的对象类型是可修改的 (mutable)，而它的原始类型是不可修改的 (immutable)。
  • 数值：整数字面量，浮点字面量。
  • 字符串
  • 布尔值：true, false
  • JS 有两个操作符用于测试两个值是否相等。一个是严格相等操作符===，如果两个值不是同一种类型，那么这个操作符就不会判定它们相等。但由于JavaScript在类型转换上很灵活，所以==操作符会做类型转换。
  • 使用 let 和 const 声明变量：let用于声明变量，const用于声明常量。
  • 解构赋值：ES6实现了一种复合声明与赋值语法，叫做解构赋值(destructuring assignment)。在解构赋值中，等号右端的值是数组或者对象(“结构化”的值)，而左端通过模拟数组或对象字面量语法指定一个或多个变量。在解构赋值发生时，会从右侧的值提取(解构)出一个或多个值，并保存到左侧列出的变量中。
  • 解构赋值可能最常用在 const, let等声明语句的初始化变量，但也可以在常规赋值表达式中使用。
