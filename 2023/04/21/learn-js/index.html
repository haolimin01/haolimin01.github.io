由来

学习JavaScript最初是因为实验室网站需要维护，之前开发的同学要毕业了，我承担了维护实验室网页的任务。后面陆陆续续学习过几次，都是了解了一些粗浅的语法内容，没有坚持下来。想着我也是可能毕业的，就把之前要做但是没有完成的东西了结一下。正好借着博客平台，督促自己记录学习过程，也提高一下我薄弱的书写水平。

原则

本着费曼学习法的原则，把复杂的东西用简单的语言描述清楚，我把这个系列教程称之为三三也能学会的JavaScript。每天学习一点点，幸福生活多一点。

准备工作

  • 参考书籍：D. Flanagan and 李松峰, JavaScript权威指南, 7th ed. 机械工业出版社, 2021.
  • 操作系统：macOS, MacBook Pro 2021 with M1 Pro Chip.
  • 浏览器：Microsoft Edge & Google Chrome.
  • 代码编辑器：Visual Studio Code & Neovim & Sublime Text.
  • JavaScript 版本：ES6

学习记录

2023/04/21, Friday

  • 参考网站：MDN, 服务器端Node.js.

  • 浏览器是JavaScript最早的宿主环境，也是JavaScript代码最常见的运行环境。2010年以后，JavaScript代码又有了另一个宿主环境，Node给与了JavaScript访问整个操作系统的权限。

  • 运行 hello world:

    console.log('hello world!')
    使用 node 运行：node hello.js
    使用浏览器运行：创建新文件 hello.html:<script src='hello.js'></script>，使用浏览器加载 hello.html: file://yourpath/hello.html
